Single-Cycle RISC-V Processor
A 32-bit single-cycle RISC-V processor designed in Verilog HDL. 
This implementation executes exactly one instruction per clock cycle using a classic single-cycle datapath based on the RV32I base integer instruction set.

🛠️ Key Features :

Architecture: 32-bit Single-Cycle Datapath
ISA Supported: RV32I subset
Supported Instructions: ALL 37 instructions of RV32I
Hardware Description Language: Verilog
